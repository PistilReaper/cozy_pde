# CozyPDE Submission Workspace

This directory stores formal task outputs that are generated by CozyPDE Agent sessions.

## Policy

- Prefer one formal autonomous session per task.
- Keep `task1_logs.log`, `task2_logs.log`, and `task3_logs.log` independent when tasks are run in separate sessions.
- Preserve Agent-generated provenance when exporting logs and packaging artifacts.
- Human operators may mechanically copy or package files, but should not manually edit generated code or predictions.

## Expected contents

- `task{N}_pred.hdf5`
- `task{N}_time.csv`
- `task{N}_logs.log`
- `code/taskN/` for task-specific generated code
- `submission.json`
- `methodology.pdf`
- `submission.zip`

## Packaging

Run `package_final` after the requested task outputs already exist. This mode validates and packages existing artifacts only; it does not rerun modeling.
